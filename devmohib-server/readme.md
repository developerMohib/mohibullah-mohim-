### How to deploy into vercel (server code)

- install vercel cli
- if you don't have installed anytime then (first time only)

```bash
npm i -g vercel 
```
- then istall vercel
```bash
npm i -g vercel@latest
```
- if installed then check version for ensure
```bash
vercel --version
```
- then command vercel prod
```bash
vercel --prod
```

### How to run 
- nodemon index.js
